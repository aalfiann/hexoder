Here is the list for 100% free image cdn, unlimited and simple to use.

    • 100% Free and Unlimited.

Conclusion

All free image CDN in this article mostly using Cloudflare CDN as backbone server. So why do we just using Cloudflare directly? Well, Using Cloudflare is not simple as we thought. Do you think as we already connected with Cloudflare DNS means we will get the cache feature? The answer is Nope, you have to know the basic knowledge about how proxy cache works properly between your server and cloudflare server, it’s really complicated to configure, if not, everything will be useless. For your information, I found many tutorial setup CDN using Cloudflare at the google search engine and 80% of them got wrong or useless. The cache seems to work normally, but actually is not. Are you able to make sure for example if the cloudflare server response 304 then the origin server return empty response? I doubt it.

Yeah that’s all for the list free image CDN. I suggest you to choose or pick one of these, because they are having pros and cons. Moreover, they are professional and already been used in many big websites.
