What is The Best Alternative MSOFFICE in Linux

2021-01-03

Hello guys, If you are new in linux world or mac, maybe you will get a little bit confusing about how to use MSOFFICE in your device. There are many of office open project out there but what is the best of it? The answer is OnlyOffice. Why and How to install it?

List Alternative MSOFFICE

  1. Online (required browser)

    • Google Docs
    • Office Online

  2. Offline

    • Libre Office
    • WPS Office
    • Calligra Office
    • OnlyOffice

I’m not talk about all the list above. Why, because I’m sure you already know about Google Docs, Libre Office, WPS Office and Calligra Office. These all project is very old. So in this article I will talk about OnlyOffice only.

Why OnlyOffice

OnlyOffice is the younger than all office project listed above. OnlyOffice first time released the desktop editor application at 2016. Even this is a younger one, but it is already mature for production use. Because this company actualy started since 2009.

I use Linux since 2009 and have already tried all office listed above. So here is my opinion why I decided to use OnlyOffice for now on.

  • Easy to Install
    OnlyOffice is very easy to install because they have native package for any Operating System.

  • Very Compatible to MSOFFICE
    This is very compatible to MSOFFICE format, doc, docx, xls, xlsx, ppt, pptx and many more. I have tried editing in both linux and windows on the same files and having no any problem.

  • Save As PDF or PDFA
    You are able to save your document directly to PDF or PDFA formatted without any plugins installed.

  • Support for Macros
    OnlyOffice have Macros support for function which is used repeatedly. This Macros is available in Words, Excel and PowerPoint.

  • Smooth, Clean and User Friendly Design
    The design is very nice, smooth, clean ad user friendly. It’s like we use browser and to open the new tab, it can be multiple format. So this is really productivity, easier and faster.

  • Having Cloud Feature
    If you already knew about 365 Office, yes OnlyOffice also have it but this feature is premium, because you are required to pay for the server cloud.

  • True Cross Platform
    OnlyOffice is the best cross platform for you to use in Linux, Mac, Android, iOS, Huawei and many more. Even if you want to deploy it through Docker or your own server, it is possible.

  • Online or Offline
    Yes, with OnlyOffice you are able to use it as offline or online. You even could colaborate with your team to working together.

  • Extra Plugins
    OnlyOffice having a plugin management feature. If you know how to code, you can just create some plugin and install it into your OnlyOffice desktop editor. Actualy, OnlyOffice have many extra plugins installed as default to help you more productivity. For example OCR, Photo Editor, Translator, Thesaurus, Send directly to email, etc.

  • Easy to Integrate
    There are many plugins to embed this OnlyOffice editor to many applications. For example, OwnCloud, Zimbra, etc. Of course you are able to integrate OnlyOffice to your website if you know how to code.

  • End to End Encryption
    At the January 2019, OnlyOffice announced the release new security an end to end encryption. This makes us feel more secured and private.

  • Licensed Under AGPL v3
    Desktop editors are distributed under AGPL v3 license for personal and commercial usage. If you are know how to code. Just improve it, the license allow you for commercial usage or just use it for personal usage.
